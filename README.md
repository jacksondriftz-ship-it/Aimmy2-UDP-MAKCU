i use a fov of 320x320 idk if its recomended but its what i use just fine

in order to run the build of aiimmy youll need .net desktop runtime 8.0.23-https://dotnet.microsoft.com/en-us/download/dotnet/8.0

to get the udp function to work youll need to first choose your model. Choose udp streaming then fill out the ip and port of your SECOND PC not the main pc thats streaming. Next you want to enable the aim assist.
Enable show detected player so a popup asking for network access show up to be able to see the stream thats incoming. Once you agree it will show the fps and latency of the stream

NOTE you will see nothing yet it will still detect and can draw boxes that you can see on your desktop, aimmy handles the stream simply as a display so the incoming stream wont be shown
